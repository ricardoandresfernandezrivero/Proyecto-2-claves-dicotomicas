-CLAVE DICOTOMICA-
El proyecto se trata de una clave dicotómica que es una especie de tabla que permite identificar plantas a partir de preguntas realizadas. 
Para este proyecto se puede realizar las preguntas y conseguir una especie de planta o también puedes bucar directamente la planta
que desea usted mediante dos metodos (Hash Table y Árbol) y que aparezcan las preguntas a realizar. Se utilizó dos JSON para dos claves distintas 
para generar la hash table y para hacer la busqueda de un elemento en el árbol.
